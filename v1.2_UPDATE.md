# 🍪 Cookie Clicker — Update Log

## v1.2 — Polish & Stability Update
*Released March 30, 2025*

### 🐛 Bug Fixes
- **Fixed broken layout on GitHub Pages** — the two-panel UI was collapsing into a
  single stacked column due to `100vh` not resolving correctly inside GitHub Pages'
  iframe environment. All panel heights have been updated to use `height: 100%` with
  `min-height: 100vh` fallbacks for full compatibility.

### ⚖️ Balance Changes
- **Removed Click Frenzy** — the Click Frenzy bonus system has been cut entirely.
  It disrupted the intended pacing and made the early game feel inconsistent.
- **Buildings now boost click power** — automation and clicking are no longer separate
  paths. Each building you own passively increases your cookies-per-click, so investing
  in infrastructure rewards both playstyles.
- **Early/mid/late game tuning pass** — the grind curve has been rebalanced. Early game
  favors active clicking, mid game opens up building synergies, and late game rewards
  players who invest in both.
